Welcome!

This Hackathon project aims to combine hardware and software tools to 
create an accisibility aid for those with hearing problems. Those who 
are hard of hearing may not be aware of loud noises in their house; the 
project seeks to cpnnect multiple sensors to detect sharp, loud noises 
and  alert the hearing impaired person that a noise was detected and 
which node it detected. It uses a Raspberry Pi as the main hub to connect 
different nodes of ESP32 microcontrollers.
